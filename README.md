# 3D-Audio


A web-based tool for experimenting with spatial audio effects. This repository provides a collection of HTML files and scripts for uploading, editing, and analyzing MP3 files with 3D audio features.

## Features

- **MP3 Upload:** Drag and drop MP3 files to get started.
- **3D Audio Effects:** Apply spatial effects like panning, reverb, and simulated environments.
- **Audio Analysis:** Visualize waveform, spectrum, and metadata.
- **Easy Customization:** Edit HTML, CSS, and JS files to tailor the experience.
- **No frameworks required:** Pure HTML/JavaScript/CSS for maximum portability.

## Usage

1. Clone or download the repository.
2. Open any `3D audio X.html` file in your browser.
3. Upload an MP3 file when prompted.
4. Use the available controls to experiment with effects and analysis.

## Installation

No installation required. Simply open the HTML files in your browser.
